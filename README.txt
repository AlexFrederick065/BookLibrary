Overview
This contract lets admin to add a book and then alot it to a user, also lets him to return the book.

Structure
- Constructor
This is used to make the address from which the contract is deployed as admin
- Struct
This contract as 1 struct that contain detail of book such as Name, Author, Book Category and Quantity Available
- Mapping
This contract has 3 mapping for efficient retrivel
- Modifier
This contract has 1 modifier that check if the address is of admin or not
- Enum
Contract has 1 Enum and it is used for containing Book Category Type

Function
- Store Book
- Get Book
- Alot Book
- Return Book
